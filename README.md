## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/Shubham0x1/REACT_Projects.git
```

Navigate into the project:

```bash
cd REACT_Projects
```

Go into any project folder and install dependencies:

```bash
npm install
```

Run the project:

```bash
npm run dev
```
